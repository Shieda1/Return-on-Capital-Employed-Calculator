What is the return on capital employed?
return on capital employed (ROCE) is a finincial ratio that can be used to assess a company's profitablility and capital efficiency. This means that the ratio can help to understand how well a company is generating profits from it's capital.
How to calculate return on capital employed?
To calculate return on capital employed you should use the following formula:
return on capital employed (ROCE) = earning before interest and tax (EBIT) / capital employed
where:
capital employed = tatal assets - current liabilities
example: company x has $135,600 total assets and $47,090 current liabilities $156,709, $30,840 EBIT, calculate ROCE of the company?
first we should calculate the capital employed
capital employed = tatal assets - current liabilities
capital employed = $135,600 - $47,090
capital employed = $88,510
and then calculate ROCE
ROCE = EBIT / capital employed
ROCE = $30,840 / $88,510
ROCE = 0.3484
Why use return on capital employed?
A company use return on capital employed to determine how profitable it is in using it's financing to generate operating income.
